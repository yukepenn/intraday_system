# data/curated/

Normalized, session-tagged, RTH-filtered parquet. Generated by `intraday data normalize ...` (Phase 1).

Curated parquet is **local-only** and **gitignored** — do not commit curated bars to Git.

## Canonical layout

```
data/curated/bars_1m_rth/asset=equity/symbol=QQQ/year=YYYY/month=MM/bars.parquet
```

## Schema

See `docs/DATA_CONTRACT.md` and `intraday.data.schema.CURATED_REQUIRED_COLUMNS` for the authoritative curated column set.

Runtime configs use **paths relative to the repo root** (e.g. `data/curated/bars_1m_rth`).
