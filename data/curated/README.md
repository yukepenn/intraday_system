# data/curated/

Normalized, session-tagged, RTH-filtered parquet. Generated by `intraday data normalize ...` (Phase 1).

## Canonical layout

```
data/curated/bars_1m_rth/asset=equity/symbol=QQQ/year=YYYY/month=MM/bars.parquet
```

## Schema (target)

Phase 1 writes these columns:

```
ts_utc, ts_local, date, session_date, session_id, bar_index, minute_of_session,
open, high, low, close, volume, source, is_rth
```

Tracking: this folder is empty until Phase 1. Curated parquet is generated and reproducible from raw; whether to track it in Git is a Phase 1 decision.
